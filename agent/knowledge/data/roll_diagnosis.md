# 侧倾问题诊断与调校

## 现象分类

侧倾（Roll）是车辆在转弯或变线时车身绕纵轴线的旋转运动。过大的侧倾会影响驾驶员信心、轮胎接地状态和操稳性能。

### 常见抱怨
- "单移线侧倾大"：快速变线时车身晃动明显，驾驶员感觉车辆不稳定。
- "转弯侧倾大"：弯道中车身外倾角度大，内侧车轮抓地力不足。
- "连续变线晃动"：连续换道时车身来回摇摆，收敛慢。

## 诊断路径

1. **确认工况**：需要明确是单移线还是转弯，车速是多少（通常 80-100 km/h）。
2. **读取基线配置**：先用 `get_current_setup` 获取当前前后稳定杆和弹簧配置。
3. **检查前后分配**：确认前后稳定杆的相对硬度关系。
4. **单变量调校**：优先改稳定杆，再考虑弹簧。

## 调校策略

### 侧倾大 + 响应慢
- 前后稳定杆整体小幅加硬（各 5%-10%）。
- 若改善不足，再考虑前轴加硬更多（因前轴通常承担更大侧倾力矩）。

### 侧倾大 + 转向不足（推头）
- 避免只加硬前稳定杆（会加重推头）。
- 可评估后稳定杆相对加硬 5%-10%。
- 同时检查前轮胎侧偏特性。

### 侧倾大 + 车尾活跃（甩尾倾向）
- 优先谨慎提高前稳定杆 5%。
- 或降低后轴滚转刚度。
- 检查后轮胎气压和定位。

## 验证指标
- 侧倾角峰值：目标是减小 10%-20%。
- 横摆角速度峰值/相位：确保横摆响应不过度延迟。
- 横向加速度 RMS：整体操稳水平。
- 主观侧倾评分：驾驶员的直接感受最为关键。

## 注意事项
- 提高滚转刚度可能增加单轮冲击，降低极限附着余量。
- 稳定杆改变会影响前后轴侧偏刚度分配，从而改变转向特性。
- 每轮只改一个轴的参数，并在同一工况下 `run_carsim` 验证。
